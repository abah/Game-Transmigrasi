Kebijakan Privasi

Transmigrasi: Membangun Negeri
Terakhir diperbarui: 9 Maret 2026
Kami sangat menghargai privasi Anda. Game ini dirancang untuk dimainkan tanpa memerlukan data pribadi apa pun.

1. Informasi yang Kami Kumpulkan

Transmigrasi: Membangun Negeri adalah game simulasi berbasis browser dan aplikasi lokal. Kami tidak mengumpulkan, menyimpan, atau memproses data pribadi pengguna.

Game ini tidak memerlukan:

  • Registrasi atau pembuatan akun
  • Nomor telepon atau alamat email
  • Informasi lokasi (GPS)
  • Akses ke kontak, kamera, atau mikrofon
  • Pembayaran atau informasi keuangan

2. Data yang Disimpan Secara Lokal

Progres permainan disimpan secara lokal di perangkat Anda menggunakan mekanisme berikut:

  • LocalStorage — untuk preferensi tampilan dan pengaturan game
  • SessionStorage — untuk data sesi aktif permainan

Data ini hanya tersimpan di perangkat Anda dan tidak pernah dikirimkan ke server manapun.

3. Layanan Pihak Ketiga

Versi web game ini dihosting menggunakan Firebase Hosting (Google LLC). Firebase Hosting dapat mencatat data teknis standar seperti alamat IP dan jenis perangkat untuk keperluan keamanan dan performa. Data ini dikelola sesuai dengan Kebijakan Privasi Google.

Untuk versi aplikasi (Android/iOS), game berjalan sepenuhnya secara offline dan tidak berkomunikasi dengan server eksternal.

4. Iklan

Game ini tidak menampilkan iklan dan tidak menggunakan jaringan iklan pihak ketiga.

5. Anak-Anak

Game ini aman untuk semua usia. Kami tidak secara sadar mengumpulkan data dari anak-anak di bawah usia 13 tahun.

6. Keamanan

Karena kami tidak mengumpulkan data pribadi, tidak ada risiko kebocoran informasi pribadi dari penggunaan game ini.

7. Perubahan Kebijakan Privasi

Jika ada perubahan pada kebijakan ini, kami akan memperbarui tanggal pada bagian atas halaman ini. Kami menyarankan untuk memeriksa halaman ini secara berkala.
